# homebrew-tap

Personal Homebrew tap for [nanjingya](https://github.com/nanjingya).

## Installation

```bash
brew tap nanjingya/tap
brew install --cask aurora-music
```

## Available Casks

| Cask | Description |
| --- | --- |
| [aurora-music](Casks/aurora-music.rb) | Aurora Music — immersive desktop music player |

## Upgrade

```bash
brew update
brew upgrade --cask aurora-music
```
